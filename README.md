# Website Đọc Truyện Tranh Online

## Giới thiệu

Website Đọc Truyện Tranh Online là dự án cá nhân nhằm xây dựng một nền tảng đọc truyện trực tuyến với giao diện đơn giản, dễ sử dụng và thân thiện với người dùng.

## Mục tiêu

* Thực hành xây dựng giao diện website
* Nâng cao kỹ năng HTML và CSS
* Áp dụng kiến thức UI/UX cơ bản vào thiết kế web

## Tính năng

* Hiển thị danh sách truyện
* Xem thông tin chi tiết truyện
* Đọc truyện theo từng chương
* Giao diện trực quan, dễ sử dụng

## Công nghệ sử dụng

* HTML
* CSS
* JavaScript
